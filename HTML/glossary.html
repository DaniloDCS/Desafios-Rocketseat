Glossário

input

A tag <input> é utilizada para a que o usuário forneça dados à aplicação. Toda via esses poderão ser utilizados para preenchimento de uma campo do formulário ou como algum argumento de uma função. Além disso, existem diversos tipo de inputs: text (Aceita todos os caracters), number (aceita apenas caracters numericos), checkbox, radio, url, email, tel, date (Uma data), submit (Submissão de formularios), entre outros. Para saber mais clique aqui e acesse a documentação oficial.

button

A tag <button></button> é utilizada, por exemplo, quando clicado, para chamar uma função ou para submissão de um formulário. Existem alguns tipo de botãos que são: button (Botão comum), submit (Dispara o evento de submissão, ou seja, o envio do formulário ), clear (limpa todos os campos de um formulário), entre outros. Para saber mais clique aqui e acesse a documentação oficial.

a

A tag <a href=""></a> é utlizada para redirecionamento, podendo ser uma ancora ou "apontar" para uma outra página. Também é utlizada para fazer downloads <a href="" download="Codando em JavaScript"></a>. Basta você referienciar o endereço de um arquivo ou site que pode ser local (em seu computador) ou extreno (na web). Para saber mais clique aqui e acesse a documentação oficial.

img

A tag <img src="" alt=""> é utlizada para adicionar uma imagem (png, jpeg, svg e afins) no seu web site. Em que o attributo src define o caminho que pode ser local ou na web CUIDADO AO UTILIZAR IMAGENS QUE POSSUEM DIREITOS AUTORAIS. E o attributo alt é como se fosse o título/descrição da imagem caso de qualquer erro na exbição da imagem por falha na conexão, erro na url ou outro qualquer. Para saber mais clique aqui e acesse a documentação oficial.
